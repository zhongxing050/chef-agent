# 厨师小D - DeepSeek Chef Agent

基于 DeepSeek API 的 AI 厨师助手，支持菜谱查询、分步做菜引导等功能。

## 项目结构

```
~/chef-agent/
├── main.py                  # 入口 + 对话循环
├── agent.py                 # Agent 核心（system prompt + API 调用）
├── tools.py                 # Agent 工具函数（检索菜谱等）
├── cookmode.py              # 做菜模式（分步引导）
├── recipes/
│   ├── __init__.py
│   ├── schema.py            # 菜谱数据结构定义
│   ├── recipes_db.py        # 菜谱数据库封装
│   └── data/
│       ├── home-cooking.json   # 家常菜（10道）
│       └── soup-stew.json      # 汤/炖菜（5道）
├── requirements.txt         # 依赖
└── README.md                # 本文件
```

## 使用方式

```bash
export DEEPSEEK_API_KEY="sk-xxx"
cd ~/chef-agent
pip install -r requirements.txt
python main.py
```

## 命令

| 命令 | 功能 |
|------|------|
| 直接输入问题或菜名 | 查询菜谱或问烹饪问题 |
| `/menu` | 查看全部菜谱 |
| `/categories` | 查看菜品分类 |
| `/cook` | 进入做菜模式（分步引导） |
| `/reset` | 重置对话 |
| `exit` | 退出 |

## 菜谱格式

每条菜谱包含完整字段：
- `id` / `name` / `category` / `difficulty`
- `prep_time` / `cook_time` / `servings`
- `description` / `tags`
- `ingredients`（名称、用量、备注）
- `steps`（顺序、描述、小贴士、预计时长）
- `tips`（全局小贴士）
